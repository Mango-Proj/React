1️⃣ React Developer Tools
React 애플리케이션의 컴포넌트 구조를 시각적으로 확인할 수 있습니다.
상태와 훅 정보를 브라우저에서 쉽게 추적하고 관리할 수 있습니다.
일반 개발자 도구로는 보기 어려운 컴포넌트 트리를 시각적으로 확인할 수 있습니다.


2️⃣ React Developer Tools 설치하기
Chrome 웹스토어에서 "React Developer Tools"를 검색하고 설치합니다.
Edge, Firefox, Safari 브라우저도 유사한 방식으로 확장 프로그램을 설치할 수 있습니다.
Electron 앱이나 NPM 패키지를 통해서도 React DevTools를 사용할 수 있습니다.



3️⃣ Components 탭
React 애플리케이션의 컴포넌트 구조를 트리 형태로 확인할 수 있습니다.
각 컴포넌트의 props, 상태, 훅 정보를 확인하고 실시간으로 값의 변화를 추적할 수 있습니다.
선택 아이콘을 클릭하고 화면의 요소를 선택하면 해당 컴포넌트를 자동으로 찾아줍니다.
콘솔 로그 아이콘을 클릭하면 선택한 컴포넌트의 정보를 브라우저 콘솔에 출력할 수 있습니다.
‘View source’ 아이콘을 클릭하면 해당 컴포넌트의 소스 코드로 바로 이동할 수 있습니다.


4️⃣ Profiler 탭
Profiler 탭은 컴포넌트 렌더링 성능을 분석하는 기능을 제공합니다.
‘Start Profiling’을 눌러 렌더링 동작을 기록할 수 있습니다.
‘Stop Profiling’을 누르면 렌더링 시간과 변경된 컴포넌트를 그래프로 확인할 수 있습니다.
느리게 렌더링되는 컴포넌트를 파악하거나 불필요한 리렌더링을 탐지할 수 있습니다.


1️⃣ React Hook Form
폼 데이터 관리를 쉽게 할 수 있는 라이브러리입니다.
사용자의 입력 값을 효율적으로 관리하고, 유효성 검사를 간단하게 구현할 수 있습니다.
Zod, Yup과 같은 검증 라이브러리와 연동이 잘 됩니다.
회원가입, 로그인 폼 같은 사용자 입력이 많은 폼에 특히 유용합니다.


2️⃣ SWR
데이터 페칭을 최적화해주는 라이브러리입니다.
캐싱, 재검증, 포커스 리페치 기능을 지원하여 실시간 데이터 갱신에 유리합니다.
서버 요청을 최소화하고, 캐시를 통해 빠른 데이터 응답이 가능합니다.
GET 요청뿐만 아니라 POST, PUT, DELETE와 같은 데이터 변경 작업도 지원합니다.
데이터 변경 후 자동으로 캐시를 갱신하거나 무효화할 수 있습니다.
UI와 서버 데이터의 일관성을 유지하기 쉽습니다.


3️⃣ Material-UI
Google Material Design을 기반으로 한 UI 컴포넌트 라이브러리입니다.
다양한 UI 컴포넌트(버튼, 네비게이션 바, 카드, 데이터 테이블 등)를 제공합니다.
테마 설정으로 전체 앱 디자인을 통일성 있게 관리할 수 있습니다.
기업용 애플리케이션에서 많이 사용됩니다.


4️⃣ Chakra UI
접근성이 뛰어난 UI 컴포넌트 라이브러리입니다.
일관된 스타일과 컴포넌트 제공으로 빠른 UI 개발이 가능합니다.
접근성 표준(ARIA)을 준수하여 다양한 사용자 환경에서 안정적인 경험을 제공합니다.
style props를 제공하여 인라인으로 빠르게 스타일링할 수 있습니다.


5️⃣ Motion
쉽고 직관적인 애니메이션 구현이 가능한 라이브러리입니다.
Motion 컴포넌트를 통해 복잡한 애니메이션도 간단하게 추가할 수 있습니다.
페이지 전환, 버튼 클릭 효과 등 시각적 효과가 중요한 프로젝트에 적합합니다.


6️⃣ Storybook
UI 컴포넌트를 독립적으로 개발하고 문서화할 수 있는 도구입니다.
컴포넌트 상태를 독립적으로 테스트하고 시각화할 수 있어 디자이너와 협업에 유리합니다.
복잡한 애플리케이션에서 컴포넌트를 체계적으로 관리할 수 있습니다.
프로젝트에 설정하고 유지보수하는 데 시간이 소요될 수 있습니다.


7️⃣ Vitest
Vite 기반의 빠른 테스트 러너입니다.
Jest와 비슷한 API를 제공해 러닝 커브가 낮습니다.
React Testing Library와 함께 사용하면 사용자 중심의 테스트를 작성할 수 있습니다.
빠른 테스트 실행이 가능하며, 개발 환경과 테스트 환경을 통합하기 좋습니다.
